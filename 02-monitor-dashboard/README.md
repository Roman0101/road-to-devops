# Infra Monitor Dashboard v1

## Descripción

Infra Monitor Dashboard es una aplicación web desarrollada para simular el monitoreo de infraestructura tecnológica. El sistema permite visualizar el estado de servidores, métricas de rendimiento y alertas operativas desde una interfaz centralizada.

Este proyecto forma parte de mi proceso de aprendizaje en las áreas de monitoreo, administración de sistemas Linux, automatización y DevOps.

---

## Características

* Dashboard principal de monitoreo.
* Visualización de servidores online y offline.
* Monitoreo de CPU, RAM y Disco.
* Sistema de alertas simuladas.
* Vista detallada de cada servidor.
* Diseño responsive para escritorio, tablet y móvil.
* Actualización de información desde archivos JSON.
* Interfaz inspirada en herramientas reales de monitoreo empresarial.
